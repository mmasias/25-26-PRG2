# Guía de desarrollo del editor de texto en consola

## 1. Definición del objetivo
Desarrollar un editor de texto sencillo que funcione en consola y permita al usuario trabajar con un número limitado de líneas mediante distintos comandos.

## 2. Estructura del programa
Organizar el programa en una única clase con un método principal, separando la lógica de cada operación en funciones independientes.

## 3. Gestión de datos
Mantener internamente una colección de líneas de texto, una referencia a la línea seleccionada y un sistema de entrada de datos por teclado.

## 4. Inicialización
Preparar la estructura de datos al inicio del programa y cargar un texto inicial que sirva como guía de uso para el usuario.

## 5. Visualización
Mostrar en cada iteración el contenido del editor, indicando la posición de cada línea y resaltando la línea seleccionada, junto con los comandos disponibles.

## 6. Interacción y operaciones
Implementar un ciclo de ejecución que permita al usuario seleccionar líneas, modificar su contenido, borrar texto, reorganizar líneas y finalizar el programa.

## 7. Control y finalización
Validar las entradas del usuario para evitar errores y permitir una salida controlada del programa.
